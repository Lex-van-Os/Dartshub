# dartsbond_almere-rare-chicken

# Project setup

## Windows

### Install packages:

```bash
pip install -r requirements.txt
```

### Install venv:

```bash
py -3 -m venv venv
```

### Activate venv:

```bash
venv\Scripts\activate
```

### Set flask in development mode:

```bash
 set FLASK_ENV=development
```

### Run the code:

```bash
flask run
```

# MacOS/Linux

### Install packages:

```bash
pip install -r requirements.txt
```

### Install venv:

```bash
python3 -m venv venv
```

### Activate venv:

Add the dot! otherwise this command won't work

```bash
. venv/bin/activate
```

### Set flask in development mode:

```bash
 set FLASK_ENV=development
```

### Run the code:

```bash
flask run
```
